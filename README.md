# windowsaloading
# Proje Açıklaması

Bu proje, Java ve Processing kullanılarak geliştirilmiş Windows tarzı bir loading screen (yükleme ekranı) uygulamasıdır.  
Amacı, basit bir arayüz animasyonu oluşturup UI geliştirme pratiği yapmaktır.

# Özellikler
- Tam ekran (fullscreen) çalışma
- Basit loading animasyonu
- Minimal Windows tarzı tasarım

# Kullanılan Teknolojiler
- Java
- Processing
- Maven
